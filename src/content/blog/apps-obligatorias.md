# 📲 Tu celular es tu mejor herramienta

Para las clases de **Applied AI** y para la logística del viaje, descarga estas apps:

* **Académico:** Microsoft Teams (para avisos) y Microsoft Project Viewer.
* **Finanzas:** **Revolut** o **Wise** (muy útiles para pagar en Libras/Euros con mejor tasa que el banco tradicional).
* **Transporte:** * **Citymapper:** (Infinitamente mejor que Google Maps para Londres y París).
    * **Trainline:** Para ver horarios de trenes hacia Hatfield.
    * **Uber / Bolt:** Para emergencias o trayectos nocturnos.
* **Traducción:** Google Translate (descarga el paquete de inglés y francés para uso offline).
